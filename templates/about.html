🎓 Dropout Prediction System

AI-powered early risk detection for student retention

📌 Problem Statement

Student dropout rates are significantly high in remote and underdeveloped regions. Early identification is difficult due to lack of data-driven systems, leading to delayed interventions and permanent dropouts.

💡 Our Solution

We developed an AI-based system that predicts student dropout risk using academic and socio-economic factors. The system provides actionable insights and suggests interventions to prevent dropouts before it happens.

🚀 Key Features

Real-time Risk Prediction
Student Performance Dashboard
Automated Intervention Suggestions
Data Visualization (Charts)
Top Risk Student Identification

⚙️ Tech Stack

FastAPI (Backend)
HTML + CSS + JS (Frontend)
Chart.js (Visualization)
Random Forest (ML Model)
SQLite / SQLAlchemy

🌍 Impact

This system enables schools and institutions to take early preventive actions, reducing dropout rates and improving educational outcomes, especially in underserved areas like rural Uttarakhand.
